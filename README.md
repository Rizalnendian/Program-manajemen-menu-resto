# Program Manajemen Menu Resto

Program Manajemen Menu Resto adalah sebuah aplikasi **Java console sederhana** yang dibuat untuk memenuhi **Tugas 3 pada mata kuliah Pemrograman Berbasis Objek/Dasar (PBD)**.
Aplikasi ini dirancang untuk membantu mengelola menu dan proses pemesanan makanan di restoran dalam lingkungan pembelajaran.
